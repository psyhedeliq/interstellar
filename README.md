## Getting Started ##

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Installing ###

1. Clone the repo  

```
git clone https://github.com/psyhedeliq/interstellar.git
```
  
2. Open home file  
  
```  
open index.html 
```
  
3. Explore  
  
```
Enjoy :)
```
